SETUP
The main computer being used in this project is a 2018 Raspberry Pi 4B with 4GB of SDRAM.
The RPi is running Debian GNU/Linux 11 "bullseye" for Raspberry Pi 6.1.21-v8+ aarch64.

On my development computer, I using a 2021 Macbook Pro laptop with macOS 14.0 "Sonoma".

With the RPi running and connected to my home Wi-Fi network, I can use Secure Shell Protocol (SSH) and
Secure Copy Protocol (SCP) to access and send files from my development computer.
